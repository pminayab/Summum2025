# 🏆 Premios Summum 2025 — Mapa de Restaurantes Lima

App web con los **50 restaurantes** premiados y nominados en los Premios Summum 2025 de Lima, Perú.

## 🚀 Publicar en GitHub Pages (3 pasos)

### 1. Crear repositorio en GitHub
1. Ir a [github.com/new](https://github.com/new)
2. Nombre sugerido: `summum2025`
3. Marcar como **Public**
4. Clic en **Create repository**

### 2. Subir el archivo
**Opción A — Desde el navegador (más fácil):**
1. En tu repositorio vacío, clic en **"uploading an existing file"**
2. Arrastra el archivo `index.html`
3. Clic en **Commit changes**

**Opción B — Con Git:**
```bash
git init
git add index.html
git commit -m "Summum 2025 mapa"
git branch -M main
git remote add origin https://github.com/TU_USUARIO/summum2025.git
git push -u origin main
```

### 3. Activar GitHub Pages
1. En tu repositorio → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / **/ (root)**
4. Clic en **Save**

⏱️ En 1-2 minutos tu app estará en:
```
https://TU_USUARIO.github.io/summum2025
```

## 📱 Características
- 50 restaurantes con info completa
- Filtros: Gran SUMMUM, Top 20, Ganadores, Nominados
- Búsqueda en tiempo real
- Teléfono, web y enlace directo a Google Maps por cada restaurante
- Indicación de si requiere reserva o no
- 100% responsive (móvil y escritorio)
- Sin dependencias externas — un solo archivo HTML

## 📊 Datos incluidos
- **Top 20** posiciones del ranking
- **Gran SUMMUM**: Astrid & Gastón, Central, Maido, Rafael

- **Nominados**: Cebicherías, Criolla, Pollerías, Sangucherías, Hamburguesas
